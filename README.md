# commercial_info_service
창업을 원하는 창업주에게 선택한 지역의 창업생태계와 유용한 정보를 제공하는 서비스.

### 안드로이드 앱 설치
commercial_info_service/Android/app/ 에서 apk를 스마트폰으로 다운 후, 실행.
